3D tool for WooCommerce which help you to configure and make your product customizable in 3D.

== Description ==

Immersive designer is Woocoomerce plugin to create a 3d configuration, including the following:
<ul>
<li>Create 3d configuration</li>
<li>Link 3d configuration to wc product</li>
</ul>
